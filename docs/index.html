Data Sources

Official APIs from government statistical offices and international organisations

Eurostat --stats europe

The statistical office of the European Union. Economy, population, trade, environment, and more across all EU member states.

~10,400 datasets SDMX 2.1 API CC BY 4.0

Swiss FSO --stats switzerland

Switzerland's official open government data portal (opendata.swiss). 288 organizations, 14 thematic categories.

~3,277 datasets CKAN API opendata.swiss Terms

UK ONS --stats uk

Office for National Statistics. UK economy, labour market, population, inflation indices, and regional data.

~337 datasets CMD API OGL v3.0

UNHCR --stats unhcr

UN Refugee Agency statistics. Refugees, asylum-seekers, IDPs, stateless persons across 249 countries since 1951.

~450 datasets Population API CC BY 4.0

HDX --stats hdx

Humanitarian Data Exchange. 400+ organizations including World Bank, WHO, WFP, UNICEF, IOM, and FAO.

~19,000+ datasets CKAN API CC BY-IGO

Netherlands --stats netherlands

Dutch national open data portal (data.overheid.nl). Government, transport, environment, and statistical datasets.

~12,245 datasets CKAN API CC0 / CC BY

Germany --stats germany

GovData.de — Germany's open government data portal. Federal, state, and municipal datasets including geospatial data.

~92,722 datasets CKAN API DL-DE-BY-2.0

Built for Serious Data Work

Production-grade reliability for bulk data collection

⚡

Async & Concurrent

Built on asyncio + httpx. Configurable parallelism with semaphore-based concurrency control.

🔄

Fully Resumable

JSON state file tracks every dataset. Stop and restart anytime without re-downloading completed data.

🛡

Rate Limit Aware

Automatic 429 detection with exponential backoff, Retry-After header support, and per-source tuning.

📊

Real-Time Progress

Rich multi-row terminal display with dataset count, transfer speed, ETA, and per-worker status.

✅

Data Integrity

--verify checks completed datasets against disk. Only marks datasets complete when actual data files are present.

🔨

Extensible Architecture

Clean DataSource interface. Add a new source in 4 files — all CLI commands work automatically.

Get Started in 30 Seconds

No pre-setup required. All directories and state files are created automatically.

# Clone and install
git clone https://github.com/compufreq/inter-stats-collector.git
cd inter-stats-collector
pip install -e .
# Start downloading
inter-collect --stats europe collect

Clean, Modular Architecture

Each source is a self-contained module implementing a common interface

cli.py --stats europe | +-> collector.py run_collection(source, ...) | +-> source.fetch_catalog(client) | +-> source.collect_datasets(catalog) | +-> filter + state check + verify | +-> async download loop (semaphore) | +-> source.download_dataset(entry) | +-> sources/ eurostat/ SDMX 2.1 API ~10,400 datasets swiss/ CKAN API ~1,977 datasets ons/ CMD API ~337 datasets unhcr/ Population API ~450 datasets hdx/ CKAN API ~244 datasets netherlands/ CKAN API ~12,245 datasets germany/ CKAN API ~92,722 datasets
