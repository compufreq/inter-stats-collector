UK ONS (Office for National Statistics)

CLI flag: --stats uk · ~337 datasets · ONS CMD API (Beta) · No authentication required

API Endpoints

Base URL: https://api.beta.ons.gov.uk/v1

EndpointURLPurpose
Dataset listingGET /datasetsPaginated list of all datasets
Dataset detailGET /datasets/{id}Full metadata
Edition listingGET /datasets/{id}/editionsEditions (e.g., "time-series")
Version detailGET /datasets/{id}/editions/{ed}/versions/{ver}Download URLs, dimensions
Downloadsdownload.ons.gov.ukCSV, XLSX, and CSV-W files

Rate Limiting

The ONS API has no documented hard rate limit but returns 429 Too Many Requests under moderate load. The collector handles this with:

  • Sequential batched version resolution (5 per batch, 1s between requests, 2s between batches)
  • Exponential backoff on 429: 20s, 40s, 80s, 160s, 320s, 640s
  • Retry-After header support
  • Concurrency capped at 2

Commands & Usage

Flag Compatibility

FlagSupportedNotes
--scope✗Not applicable — ONS is a single source
--formats✗Not applicable — always downloads CSV + XLSX + CSV-W
--year-from / --year-to✗Not applicable — each dataset contains all available years
--concurrency✓Auto-capped at 2 (ONS rate limits aggressively)
--verify✓Checks that CSV/XLSX/CSV-W files exist on disk (not just metadata)
--dry-run✓Preview without downloading
--retry-failed✓Retry datasets that failed due to 429 errors
--filter-path✓Filter by taxonomy path (e.g., economy/inflationandpriceindices)
--filter-codes✓Filter by dataset codes (e.g., cpih01,mm23)
--folder-style✓display (slugified titles) or code (raw codes)

Quick Reference (All Flags)

# collect — all applicable flags for UK ONS
inter-collect --stats uk collect \
  -o <OUTPUT_DIR> \
  -c <CONCURRENCY:int> \
  -d <DELAY:seconds> \
  --skip-existing | --no-skip-existing \
  --retry-failed \
  --filter-path <TAXONOMY_PATH:str> \
  --filter-codes <CODE1,CODE2,...> \
  --dry-run \
  --verify \
  --folder-style <display|code> \
  -v

# status — show collection progress
inter-collect --stats uk status -o <OUTPUT_DIR>

# tree — display category hierarchy
inter-collect --stats uk tree --depth <DEPTH:int> -v

# rename — switch folder naming style
inter-collect --stats uk rename <display|code> -o <OUTPUT_DIR> --dry-run -v

Not applicable for UK ONS: --scope, --formats, --year-from, --year-to (silently ignored)

Note: --concurrency is auto-capped at 2 regardless of the value passed.

Download Datasets

# Download all ~337 ONS datasets
inter-collect --stats uk collect

# Preview what would be downloaded
inter-collect --stats uk collect --dry-run

# Download specific datasets
inter-collect --stats uk collect --filter-codes cpih01,mm23,lms

# Retry previously failed datasets (common after 429 rate limiting)
inter-collect --stats uk collect --retry-failed

# Verify completed datasets have actual data files (not just info/meta JSON)
# This catches datasets that were marked complete but only have metadata
# due to 429 errors during the original download
inter-collect --stats uk collect --verify

Important: ONS concurrency is automatically capped at 2 regardless of what you pass with --concurrency. This is because the ONS API returns 429 Too Many Requests under even moderate load.

Check Collection Status

inter-collect --stats uk status

View Category Tree

inter-collect --stats uk tree
inter-collect --stats uk tree --depth 4

Rename Folders

inter-collect --stats uk rename code --dry-run
inter-collect --stats uk rename display

Files Per Dataset

For a dataset with code cpih01, up to 5 files are created:

FileFormatDescription
cpih01_info.jsonJSONCatalog snapshot
cpih01_meta.jsonJSONFull version metadata (dimensions, release info)
cpih01.csvCSVData in CSV format
cpih01.xlsxXLSXData in Excel format
cpih01.csv-metadata.jsonJSONCSV-W metadata (data dictionary)
