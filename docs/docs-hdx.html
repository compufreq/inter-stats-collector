HDX (Humanitarian Data Exchange)

CLI flag: --stats hdx · ~244 (UNHCR) / ~19,000+ (all orgs) datasets · HDX CKAN API · No authentication required for reads

About HDX

The Humanitarian Data Exchange (HDX) is OCHA's open platform for sharing humanitarian data across crises and organisations. It is built on CKAN and hosts data from 481 organizations including UN agencies, NGOs, governments, and research institutions.

HDX complements the UNHCR Statistics API by providing per-country refugee population CSVs, situation reports, border crossing data, camp locations, and voluntary repatriation statistics — the country-level breakdown data that the UNHCR Statistics API does not expose publicly.

Commands & Usage

Flag Compatibility

FlagSupportedNotes
--scope✓Organization filter: slug or all (default: UNHCR)
--formats✓Comma-separated formats (default: csv,xlsx,xls)
--year-from / --year-to✗Not applicable — UNHCR source only
--concurrency✓Default 5; HDX has no documented rate limits
--verify✓Checks that data files (CSV/XLSX/XLS) exist on disk
--dry-run✓Preview without downloading
--retry-failed✓Retry previously failed datasets
--filter-path✓Filter by group path
--filter-codes✓Filter by CKAN package names
--folder-style✓display (slugified) or code (raw CKAN names)

Quick Reference (All Flags)

# collect — all applicable flags for HDX
inter-collect --stats hdx \
  --scope <ORG_SLUG:str|all> \
  --formats <csv,xlsx,xls> \
  collect \
  -o <OUTPUT_DIR> \
  -c <CONCURRENCY:int> \
  -d <DELAY:seconds> \
  --skip-existing | --no-skip-existing \
  --retry-failed \
  --filter-path <GROUP_PATH:str> \
  --filter-codes <PACKAGE1,PACKAGE2,...> \
  --dry-run \
  --verify \
  --folder-style <display|code> \
  -v

# status — show collection progress
inter-collect --stats hdx status -o <OUTPUT_DIR>

# tree — display category hierarchy
inter-collect --stats hdx tree --depth <DEPTH:int> -v

# rename — switch folder naming style
inter-collect --stats hdx rename <display|code> -o <OUTPUT_DIR> --dry-run -v

Not applicable for HDX: --year-from, --year-to (silently ignored)

--scope (Organization Filter)

Use --scope to select which organization's datasets to download. Accepts any CKAN organization slug.

  • (default): UNHCR datasets only (~244 with downloadable CSV/XLS)
  • --scope all: All 364+ organizations (~16,390 datasets)
  • --scope ORG_SLUG: Any specific organization

Browse all HDX organizations to find the right scope value.

--formats

  • Default: csv,xlsx,xls (all downloadable data formats)
  • Skips PDF, Web App, KMZ, and other non-data resources

Download Datasets

# Download all UNHCR datasets (default)
inter-collect --stats hdx collect

# Preview what would be downloaded
inter-collect --stats hdx collect --dry-run

# Verify completed datasets have actual data files, re-download if missing
inter-collect --stats hdx collect --verify

# World Food Programme
inter-collect --stats hdx --scope wfp collect

# International Organization for Migration
inter-collect --stats hdx --scope international-organization-for-migration collect

# World Health Organization, CSV only
inter-collect --stats hdx --scope world-health-organization --formats csv collect

# All organizations
inter-collect --stats hdx --scope all collect

# Download specific datasets by CKAN package name
inter-collect --stats hdx collect --filter-codes unhcr-population-data-for-afg,unhcr-situations

# Retry previously failed datasets
inter-collect --stats hdx collect --retry-failed

Check Collection Status

inter-collect --stats hdx status

View Category Tree

inter-collect --stats hdx tree
inter-collect --stats hdx tree --depth 3

Rename Folders

inter-collect --stats hdx rename code --dry-run
inter-collect --stats hdx rename display

CC BY-SAShare-alike — derivatives must use same license

Check each dataset's metadata for its specific license. The _info.json file includes license_url and license_title fields.
