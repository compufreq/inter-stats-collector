Eurostat (EU)

CLI flag: --stats europe · ~10,400 datasets · SDMX 2.1 Dissemination API · No authentication required

API Endpoints

Base URL: https://ec.europa.eu/eurostat/api/dissemination

EndpointURLPurpose
Catalogue/catalogue/toc/xmlFull Table of Contents (XML)
SDMX Data/sdmx/2.1/data/{code}Statistical data (TSV + SDMX XML)
Datastructure/sdmx/2.1/datastructure/ESTAT/{code}/latestData Structure Definition
Dataflow/sdmx/2.1/dataflow/ESTAT/{code}/1.0Dataflow with resolved codelists
Content Constraint/sdmx/2.1/contentconstraint/ESTAT/{code}/latestValid dimension values
ESMS Metadataec.europa.eu/eurostat/cache/metadata/en/{ref}_esms.htmMethodology, quality reports

Commands & Usage

Flag Compatibility

FlagSupportedNotes
--scope✗Not applicable — Eurostat has a single source
--formats✗Not applicable — always downloads TSV + SDMX + structural metadata
--year-from / --year-to✗Not applicable — each dataset contains all available years
--concurrency✓Default 5; safe to increase for Eurostat
--verify✓Checks that TSV or SDMX data files exist on disk
--dry-run✓Preview without downloading
--retry-failed✓Retry datasets that failed in a previous run
--filter-path✓Filter by catalog path (e.g., database_by_themes/economy)
--filter-codes✓Filter by dataset codes (e.g., prc_hicp_midx,ei_bsin_q_r2)
--folder-style✓display (human-readable) or code (raw codes)

Quick Reference (All Flags)

# collect — all applicable flags for Eurostat
inter-collect --stats europe collect \
  -o <OUTPUT_DIR> \
  -c <CONCURRENCY:int> \
  -d <DELAY:seconds> \
  --skip-existing | --no-skip-existing \
  --retry-failed \
  --filter-path <CATALOG_PATH:str> \
  --filter-codes <CODE1,CODE2,...> \
  --dry-run \
  --verify \
  --folder-style <display|code> \
  -v

# extract — decompress .tsv.gz and .sdmx.xml.gz files
inter-collect --stats europe extract \
  -o <OUTPUT_DIR> \
  -c <WORKERS:int> \
  -b <BUFFER_MB:int> \
  --force \
  -v

# status — show collection progress
inter-collect --stats europe status -o <OUTPUT_DIR>

# tree — display category hierarchy
inter-collect --stats europe tree --depth <DEPTH:int> -v

# rename — switch folder naming style
inter-collect --stats europe rename <display|code> -o <OUTPUT_DIR> --dry-run -v

Not applicable for Eurostat: --scope, --formats, --year-from, --year-to (silently ignored)

Download Datasets

# Download all ~10,400 datasets (default)
inter-collect --stats europe collect

# Preview what would be downloaded
inter-collect --stats europe collect --dry-run

# Download with higher concurrency
inter-collect --stats europe collect --concurrency 10

# Download specific datasets only
inter-collect --stats europe collect --filter-codes prc_hicp_midx,nama_10_gdp

# Download only datasets under a category path
inter-collect --stats europe collect --filter-path database_by_themes/economy

# Retry previously failed datasets
inter-collect --stats europe collect --retry-failed

# Verify completed datasets have actual data files, re-download if missing
inter-collect --stats europe collect --verify

Extract Compressed Files

Eurostat delivers data as .tsv.gz and .sdmx.xml.gz. After collection, extract them:

# Extract all .gz files (auto-detects CPU cores for parallelism)
inter-collect --stats europe extract

# Extract with specific worker count and buffer size
inter-collect --stats europe extract --concurrency 8 --buffer 128

# Re-extract already extracted files
inter-collect --stats europe extract --force

Check Collection Status

inter-collect --stats europe status

View Category Tree

# Show category tree (default depth 3)
inter-collect --stats europe tree

# Show deeper nesting
inter-collect --stats europe tree --depth 5

Rename Folders

# Switch from display names to raw codes
inter-collect --stats europe rename code --dry-run
inter-collect --stats europe rename code

# Switch back to display names
inter-collect --stats europe rename display

Disk Space

A full Eurostat collection requires approximately 150-200 GB:

ComponentSize
Compressed data (.tsv.gz + .sdmx.xml.gz)~30-40 GB
Extracted data (.tsv + .sdmx.xml)~100-120 GB
Structural metadata (DSD + Dataflow + Constraint)~15-20 GB
Reference metadata + Info JSON~1-2 GB
