CLI Documentation

Complete reference for the inter-collect command-line tool.

Commands

CommandDescription
collectDownload datasets from selected source(s)
extractDecompress .gz files in parallel
statusShow collection progress and disk statistics
treeDisplay the category/taxonomy tree
renameSwitch folder naming between display and code styles

Global Flags

These flags are placed before the command name and apply to all sources.

FlagDefaultDescription
--statsallSource: all, europe, uk, switzerland, unhcr, hdx
-o, --output./stats_dataRoot output directory
-v, --verboseoffEnable debug logging

Source-Specific Flags

These flags are also placed before the command but only affect certain sources. They are silently ignored by other sources.

FlagSourcesDescription
--scope <ORG>Swiss, HDXOrganization filter: slug or all
--formats <FMT,...>Swiss, HDXComma-separated data formats (e.g., csv,xls)
--year-from <YEAR>UNHCRStart year (default: 1951)
--year-to <YEAR>UNHCREnd year (default: latest)

collect

Download all datasets from the selected source(s).

inter-collect [--stats <SOURCE>] [source-flags] collect [collect-flags]
FlagDefaultDescription
-c, --concurrency5Max parallel downloads (auto-capped for ONS=2, UNHCR=1)
-d, --delay0.5Seconds between download batches
--skip-existingonSkip already downloaded files
--no-skip-existing—Re-download all files
--retry-failedoffInclude previously failed datasets
--filter-path <PATH>—Only download under this catalog path
--filter-codes <A,B,...>—Only download these dataset codes
--dry-runoffPreview without downloading
--verifyoffCheck completed datasets for missing data files and re-download
--folder-styledisplaydisplay (human-readable) or code (raw codes)

extract

Decompress .tsv.gz and .sdmx.xml.gz files using parallel workers. Primarily used for Eurostat.

inter-collect [--stats <SOURCE>] extract [extract-flags]
FlagDefaultDescription
-c, --concurrencyautoWorker count (default: 2x CPU cores, max 32)
-b, --buffer64I/O buffer size in MB per worker
--forceoffRe-extract already extracted files

status

Show collection progress, state file summary, and disk statistics.

inter-collect [--stats <SOURCE>] status

No additional flags — uses -o to find the data directory.

tree

Fetch and display the category/taxonomy tree from the data source.

inter-collect [--stats <SOURCE>] tree [--depth <N>]
FlagDefaultDescription
--depth3Maximum tree depth to display

rename

Rename existing directories between naming styles.

inter-collect [--stats <SOURCE>] rename <display|code> [--dry-run]
Argument/FlagDescription
displayRename folders to human-readable slugified titles
codeRename folders to raw statistical office codes
--dry-runPreview what would be renamed without applying

Common Patterns

Download everything from all sources

inter-collect collect

Preview before downloading

inter-collect --stats europe collect --dry-run

Verify and fix incomplete downloads

inter-collect --stats uk collect --verify

Retry failed datasets

inter-collect --stats switzerland collect --retry-failed

Download specific datasets

inter-collect --stats europe collect --filter-codes prc_hicp_midx,nama_10_gdp

Swiss: specific organization, CSV only

inter-collect --stats switzerland --scope bundesamt-fur-energie-bfe --formats csv collect

HDX: World Food Programme data

inter-collect --stats hdx --scope wfp collect

UNHCR: recent years only

inter-collect --stats unhcr --year-from 2020 --year-to 2024 collect

Extract compressed Eurostat data

inter-collect --stats europe extract --concurrency 8 --buffer 128

Check collection status

inter-collect status
