Swiss FSO (opendata.swiss)

CLI flag: --stats switzerland · ~1,977 (BFS) / ~6,699 (all orgs) datasets · opendata.swiss CKAN API · No authentication required

Commands & Usage

Flag Compatibility

FlagSupportedNotes
--scope✓Organization filter: slug or all (default: BFS)
--formats✓Comma-separated formats (default: csv,xls,ods,json)
--year-from / --year-to✗Not applicable — UNHCR only
--concurrency✓Default 5; no aggressive rate limiting
--verify✓Checks that data files (CSV/XLS/ODS/JSON) exist on disk
--dry-run✓Preview without downloading
--retry-failed✓Retry previously failed datasets
--filter-path✓Filter by group path
--filter-codes✓Filter by CKAN package names
--folder-style✓display (slugified) or code (raw CKAN names)

Quick Reference (All Flags)

# collect — all applicable flags for Swiss FSO
inter-collect --stats switzerland \
  --scope <ORG_SLUG:str|all> \
  --formats <csv,xls,ods,json> \
  collect \
  -o <OUTPUT_DIR> \
  -c <CONCURRENCY:int> \
  -d <DELAY:seconds> \
  --skip-existing | --no-skip-existing \
  --retry-failed \
  --filter-path <GROUP_PATH:str> \
  --filter-codes <PACKAGE1,PACKAGE2,...> \
  --dry-run \
  --verify \
  --folder-style <display|code> \
  -v

# status — show collection progress
inter-collect --stats switzerland status -o <OUTPUT_DIR>

# tree — display category hierarchy
inter-collect --stats switzerland tree --depth <DEPTH:int> -v

# rename — switch folder naming style
inter-collect --stats switzerland rename <display|code> -o <OUTPUT_DIR> --dry-run -v

Not applicable for Swiss FSO: --year-from, --year-to (silently ignored)

--scope (Organization Filter)

Use --scope to select which organization's datasets to download. Accepts any opendata.swiss organization slug.

  • (default): BFS only — bundesamt-fur-statistik-bfs (~3,277 datasets)
  • --scope all: All 288 organizations (~6,700+ datasets with downloadable resources)
  • --scope ORG_SLUG: Any specific organization

Browse all Swiss organizations to find the right scope value.

--formats

  • Default: csv,xls,ods,json (all downloadable data formats)
  • Example: --formats csv for CSV only (~308 datasets)
  • Skips HTML, SERVICE, PDF, and other non-data resources automatically

Download Datasets

# Download all BFS datasets, all formats (default)
inter-collect --stats switzerland collect

# Preview what would be downloaded
inter-collect --stats switzerland collect --dry-run

# Verify completed datasets have actual data files, re-download if missing
inter-collect --stats switzerland collect --verify

# CSV only
inter-collect --stats switzerland --formats csv collect

# All organizations
inter-collect --stats switzerland --scope all collect

# Specific org: swisstopo
inter-collect --stats switzerland --scope bundesamt-fur-landestopografie-swisstopo collect

# Specific org: Federal Office of Energy, CSV only
inter-collect --stats switzerland --scope bundesamt-fur-energie-bfe --formats csv collect

# Download specific datasets by CKAN package name
inter-collect --stats switzerland collect --filter-codes rohholzbilanz3,erwerbstatige-inlandkonzept

# Retry previously failed datasets
inter-collect --stats switzerland collect --retry-failed

Check Collection Status

inter-collect --stats switzerland status

View Category Tree

inter-collect --stats switzerland tree
inter-collect --stats switzerland tree --depth 4

Rename Folders

inter-collect --stats switzerland rename code --dry-run
inter-collect --stats switzerland rename display

When multiple resources of the same format exist in different languages, filenames include a language suffix (e.g., dataset_de.csv, dataset_fr.csv).
